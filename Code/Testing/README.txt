1. Download the DEMO folder

2. Setup the anaconda environment using the 'working1.yml' file included 

3. From the command propmpt, activate the environment 
	- conda activate working1

4. Open the directory "DEMO"
	- cd "C:\Users\[insert]\Downloads\DEMO" 

5. Run the toolbox script
	- python demo_toolbox.py

6. Once the toolbox opens up, Click the Browse button and select the "sample" audio file included

7. Enter the text you want to synthesize in the text box and click synthesize and vocode
